# bashllm

## Requirements
`rust`

## Installation

Clone this repo:
```
git clone https://github.com/noahshinn024/bashllm
```

Run `make`:
```
cd bashllm && make
```

## To Run

```
bashllm <command description>
```

Example:
```
>>> bashllm "a command that echoes something"
echo "hello world"
```
