# BashLLM
Faster search for shell commands.

## Requirements
`rust`

## Installation

Clone this repo:
```
git clone https://github.com/noahshinn024/bashllm
```

Run `make`:
```
cd ./bashllm && make
```

## To Run

```
bashllm <command description>
```

Example:
```
>>> bashllm "search for all open ports on ip"
echo "sudo nmap <ip addr>"
```
